# E-Commerce API with FastAPI

## Description
A full-stack E-commerce API built with FastAPI and MySQL. Features include user authentication, product management, and database integration.

## Technologies
- Python
- FastAPI
- MySQL
- SQLAlchemy
- Pydantic

## Features
- User registration and login
- JWT Authentication
- CRUD operations for products
- Database integration with MySQL
